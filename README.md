💰 Transparent Donation Tracker dApp

A simple and transparent blockchain-based donation tracking system built using Solidity, Hardhat, and Ethers.js.

🚀 Project Overview

This decentralized application (dApp) allows users to:

Donate ETH securely
View all donations publicly
Ensure transparency using blockchain
Track total contributions and top donors
🧠 How It Works
Users connect their wallet using MetaMask
Donations are sent to a smart contract
Each transaction is stored on the blockchain
Anyone can view donation history
🛠️ Tech Stack
Solidity (Smart Contract)
Hardhat (Development Environment)
Ethers.js (Blockchain Interaction)
HTML, CSS, JavaScript (Frontend)
MetaMask (Wallet Integration)

⚙️ Setup Instructions
1️⃣ Clone the repository
git clone SWATHI
cd donation-dapp
2️⃣ Install dependencies
npm install
3️⃣ Start local blockchain
npx hardhat node
4️⃣ Deploy contract
npx hardhat run scripts/deploy.js --network localhost
5️⃣ Run frontend

Open:

frontend/index.html
